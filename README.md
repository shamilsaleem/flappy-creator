
# flappy-creator

A simple parody flappy bird game creator using Node.js and SQLite.

A fork of [hyspace/flappy](https://github.com/hyspace/flappy)

## Run your own

```bash
git clone https://github.com/shamilsaleem/flappy-creator.git
cd flappy-creator
npm install
npm start
```
